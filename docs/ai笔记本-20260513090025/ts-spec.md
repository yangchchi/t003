## 1. 技术栈
- 后端：Node.js (TypeScript), NestJS
- 前端：React (TypeScript), Next.js (App Router)
- 数据库：PostgreSQL (主数据), Redis (会话/缓存)

## 2. 数据模型设计
```sql
-- 用户表
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE,
  phone VARCHAR(20) UNIQUE,
  password_hash VARCHAR(255) NOT NULL,
  nickname VARCHAR(100) NOT NULL,
  avatar_url VARCHAR(500),
  is_guest BOOLEAN DEFAULT FALSE,
  guest_expires_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 笔记本表
CREATE TABLE notebooks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  name VARCHAR(200) NOT NULL,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(user_id, name)
);

-- 笔记表
CREATE TABLE notes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  notebook_id UUID REFERENCES notebooks(id) ON DELETE SET NULL,
  title VARCHAR(500),
  content TEXT NOT NULL,
  content_hash VARCHAR(64) NOT NULL, -- 用于冲突检测
  version INTEGER DEFAULT 1,
  last_modified_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 标签表
CREATE TABLE tags (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  name VARCHAR(100) NOT NULL,
  is_auto_generated BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(user_id, name)
);

-- 笔记-标签关联表
CREATE TABLE note_tags (
  note_id UUID NOT NULL REFERENCES notes(id) ON DELETE CASCADE,
  tag_id UUID NOT NULL REFERENCES tags(id) ON DELETE CASCADE,
  PRIMARY KEY (note_id, tag_id)
);

-- 同步操作记录表
CREATE TABLE sync_operations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  device_id VARCHAR(100) NOT NULL,
  entity_type VARCHAR(50) NOT NULL, -- 'note', 'notebook', 'tag'
  entity_id UUID NOT NULL,
  operation VARCHAR(20) NOT NULL, -- 'create', 'update', 'delete'
  data JSONB,
  timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  processed BOOLEAN DEFAULT FALSE
);

-- 冲突备份表
CREATE TABLE conflict_backups (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  original_note_id UUID NOT NULL REFERENCES notes(id) ON DELETE CASCADE,
  title VARCHAR(500) NOT NULL,
  content TEXT NOT NULL,
  conflict_timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  resolved_at TIMESTAMP
);
```

## 3. API设计

### 认证相关
- 路径：`POST /api/auth/register`
- 请求：
```typescript
interface RegisterRequest {
  email?: string;
  phone?: string;
  password: string;
  nickname: string;
  verificationCode: string; // 短信或邮箱验证码
}
```
- 响应：
```typescript
interface AuthResponse {
  user: {
    id: string;
    email?: string;
    phone?: string;
    nickname: string;
    avatar_url?: string;
  };
  accessToken: string;
  refreshToken: string;
}
```
- 错误码：`4001` 验证码错误, `4002` 用户已存在, `4003` 参数格式错误

### 笔记管理
- 路径：`POST /api/notes`
- 请求：
```typescript
interface CreateNoteRequest {
  notebookId?: string;
  title?: string;
  content: string;
}
```
- 响应：
```typescript
interface NoteResponse {
  id: string;
  title?: string;
  content: string;
  contentHash: string;
  version: number;
  notebookId?: string;
  tags: Array<{
    id: string;
    name: string;
    isAutoGenerated: boolean;
  }>;
  outline?: Array<{
    level: number;
    text: string;
    anchor: string;
  }>;
  relatedNotes?: Array<{
    id: string;
    title?: string;
    preview: string;
  }>;
  createdAt: string;
  updatedAt: string;
}
```
- 错误码：`4101` 笔记本不存在, `4102` 内容为空

### 同步接口
- 路径：`POST /api/sync/push`
- 请求：
```typescript
interface SyncPushRequest {
  deviceId: string;
  operations: Array<{
    entityType: 'note' | 'notebook' | 'tag';
    entityId: string;
    operation: 'create' | 'update' | 'delete';
    data?: any;
    timestamp: string;
  }>;
}
```
- 响应：
```typescript
interface SyncPushResponse {
  conflicts: Array<{
    entityType: string;
    entityId: string;
    resolvedVersion: any;
    backupId?: string;
  }>;
  latestTimestamp: string;
}
```
- 错误码：`4201` 冲突检测失败

### AI服务
- 路径：`POST /api/ai/process-note`
- 请求：
```typescript
interface ProcessNoteRequest {
  noteId: string;
```

- 响应：
```typescript
interface ProcessNoteResponse {
  tags: string[];
```

- 错误码：`4301` AI处理超时, `4302` 内容过长

### 访客体验
- 路径：`POST /api/guest/speech-to-text`
- 请求：`FormData` (audio: Blob)
- 响应：
```typescript
interface SpeechToTextResponse {
  text: string;
  confidence: number;
  segments: Array<{
    text: string;
    confidence: number;
    start: number;
    end: number;
  }>;
  remainingSeconds: number; // 剩余体验时长
}
```
- 错误码：`4401` 体验次数超限, `4402` 音频格式不支持

## 4. 异常处理
- 错误规则：
  1. 输入验证失败 → 400 + 具体字段错误
  2. 认证失败 → 401
  3. 权限不足 → 403
  4. 资源不存在 → 404
  5. 业务规则违反 → 409 + 错误码
  6. 服务器错误 → 500 + 错误ID（用于追踪）
  7. 第三方服务失败 → 502 + 服务名称
